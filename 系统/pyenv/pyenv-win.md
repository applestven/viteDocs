## 安装

1. git clone https://github.com/pyenv-win/pyenv-win.git
2. 修改环境变量
**添加PYENV添加到系统变量，而不是用户变量。**
变量名：PYENV
变量值：C:\Users\用户名.pyenv\pyenv-win 这里填写你的pyenv路径

**添加PATh变量**
%PYENV%\bin
%PYENV%\shims
